HTML BÁSICO

Pratica de estudos HTML básico.

O objetivo é externar o que vem sendo aprendido em HTML. Abaixo irei apresentar algumas tags e suas funcionalidades.

As tags <h1></h1> .. <h6><h/6> indicam níveis de titulos indo do nível 1 até o nível 6.

Exemplo:

Nível 02

Nível 06

A tag <i></i> é usada para colocar uma palavra ou texto em italíco como apresentado.

A tag <strong></strong> é usada para deixar uma palavra ou texto em negrito como aprensentado.

A tag <br> é usada para uma quebra linha.
